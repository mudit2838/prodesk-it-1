The project focuses on frontend fundamentals including responsive layouts, DOM manipulation, accessibility, UI interactions, and performance optimization using only HTML, CSS, and Vanilla JavaScript.

# ⚙️ Features

## Responsive Design
The landing page is fully responsive across:
- Desktop
- Tablet
- Mobile devices

---

## Dark / Light Mode
Implemented using Vanilla JavaScript by toggling a CSS class on the `<body>` element.

---

## Sticky Glassmorphism Navbar
The navbar remains fixed during scrolling and uses a frosted glass blur effect.

---

## Accessibility Optimizations
Implemented accessibility improvements including:
- Semantic button elements
- ARIA labels
- Keyboard focus states
- Accessible navigation structure

---

## SEO Optimization
Includes:
- Proper heading hierarchy
- Meta description
- Crawlable navigation links
- Semantic HTML5 structure
